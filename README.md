# Projects

This folder contains a small collection of web projects and demos built with HTML, CSS, and JavaScript. Each subfolder showcases a different example, such as a calculator, a product landing page, pet care content, and simple HTML layout examples.

## Included Projects

- `calculator/` — a basic web calculator application with its own HTML, CSS, and JavaScript files.
- `Nike/` — a simple product landing page example with Nike-themed styling and imagery.
- `petcare/` — a pet care/web content page with supporting styles and images.
- `Html/` — sample HTML pages demonstrating basic form and heading layouts with supporting CSS.

## How to use

Open any project folder in a browser to view the live example. Each project is self-contained and can be edited directly for learning or experimentation.
